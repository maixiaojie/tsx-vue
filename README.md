# tsx-vue-starter

vue + vue-composition-api + TypeScript + tsx + vue-router + Vuex 项目实战

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
